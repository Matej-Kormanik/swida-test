## Running the app

### Backend
1. Navigate to the py-rest folder
2. `python3 manage.py makemigrations`
3. `python3 manage.py migrate`
4. `python3 manage.py runserver`

### Web
1. Navigate to the vue-web folder
2. `npm install`
3. `npm run dev`
